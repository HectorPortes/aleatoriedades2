Configurações git via VSCode

Passo 1: abrir o terminal
Passo 2: git init
Passo 3: git config --global user.name "meu nome (do github)"
Passo 4: git config --global user.email "meu email (do github)"
Passo 5: git remote add oriin "https://github.com/nome_do_repositorio_do_github.git"
Passo 6: git branch -M master

Upload de arquivos para o github

Passo 1: git add .
Passo 2: git commit -m "lembrete de upload"
Passo 3: git push --set-upstream origin master (isso só pela primeira vez)
